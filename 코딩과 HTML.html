코딩과 HTML

    1.코딩
    2.html
1)코딩
  • 코딩 = code+ing 합성어 = 코드(code)를 작성하다(ing)는 의미
    • -코드는 "컴퓨터가 알아들을 수 있는 언어(기계어)"로 컴퓨터에게 명령함
    -사람이 하는 일(원인)
    부호 또는 신호라는 의미의 코드 (code),
    원천이라는 뜻의 소스 (Source),
    약속이라는 의미에서 언어 그 중에서 컴퓨터 언어라고 함

    2)html


  • Hyper Text Markup Language의 약자 = 하이퍼텍스트로 구성+구조를 표시하는+언어

  • 웹 문서를 만들기 의해 약속된 언어

    • -기계가 하는 일(결과)
    애플리케이션(줄여서) 앱,
    (한국어로) 응용 프로그램,
    프로그램
    웹에서는 결과를 웹 페이지,
    웹페이지가 모여있으면 웹사이트라고 함.
    기능이 좀 많으면 웹애플리케이션이라고 함.
    줄여서 웹앱이라고도 함

  • 텍스트 문서 만들고 웹페이지에서 실행하기 위해서 html웹문서로 인식을 할 수 있도록 뒤에 .html붙이기

  • html문서 기본 태그
