-HTML이라는 이미 있는 언어의 디자인과 관련된 새로운 태그로 추가하는 방법
> 디자인을 위한 태그 font 탄생함

3)CSS 사용법
첫번째-style태그
:style태그는 HTML의 문법이면서 style태그 안쪽에 있는 내용을 CSS언어의 문법에 맞게 해석해서 처리하게 함
ex)style
r {
color:red;
}
/style
두번째-속성을 이용하는 방법
:html태그 안에 직접 "style"을 사용하기
(html과 CSS의 사용 설명서-style속성 사용시 그 속성값을 웹브라우저는 CSS의
문법에 맞게 해석해서 그결과를 style속성이 위치하는 태그에 적용할 것)
b style="color:blue" /b

현재 선택되어 있는 웹페이지 만큼은 밑줄을 긋는 방법
b style="color:blue;text-decoration:underline" /b

tip)텍스트에 꾸밈,장식(decoration) 없애는 방법 > style태그 안에 text-decoration: none; 추가하기
